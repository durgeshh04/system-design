# Polymorphism:

- Polymorphism is a important fundamental concept of oop.
- It allows object of the different classes to be treated as object of the common superclass.

## In Typescript we can acheive polymorphism in three ways.

1. Sub-Type Polymorphism:

- Verdict: Sub-Type Polymorphism (also known as Inclusion Polymorphism or Method Overriding). You can substitute a sub-type object where a super-type object is expected, and the system executes the specialized sub-type behavior.

2. Parametric Polymorphism:

- Verdict: This is the definition of Parametric Polymorphism in TypeScript, achieved entirely through Generics. It allows you to write functions or classes that work uniformly over different types, ensuring type safety without sacrificing code reuse.

3. Ad-hoc Polymorphism:

- This also known as overloading or coersion polymorphism.
